Gerador de Senha - Projetinho pessoal para fins de estudos

O projeto é um gerador de senhas seguras utilizando Java, Spring, HTML, CSS, E JavaScript. Este projeto é uma aplicação full-stack de gerador de senhas, utilizando Java com Spring Boot no backend e HTML, CSS, E JavaScript no frontend.
